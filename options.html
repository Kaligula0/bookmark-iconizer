Damn! There are soo many steps here!

Yes, you may think so. But that's very easy to do, when most things are automated by this Chrome extension :)
However, not everything can be handled by the extension because Chrome's policy disallows uploading, downloading, editing user files from disk drive.
And your job is only to choose bookmarks to change, choose icons for them, then save and import to Chrome ;)
Thanks to Rod Hilton for giving the idea and finding this wonderful import/export solution!

(Have any questions? Problems? Requests? Report it on GitHub.)
(Like this extension on official Facebook page!)

(Remember – the extension doesn't collect *any* of your data.)

Let's do this!

1. Choose bookmarks you want to iconize

Search for your bookmarks you want to iconize (you may search by title or URI address; type in at least 3 characters why?) then add to worklist by clicking green "+" sign at the end of the row:

Automatic favicons are provided by Google, if your search returns too many results be aware that some of the icons my not be loaded due to query overload.
The icon means Google couldn't determine website's favicon, probably because the website doesn't use one.
Text (JS) instead of icon means that the link (bookmark) is a bookmarklet, which means it's not an URI to any website but a JavaScript code which of course is not related to any website's favicon.

Search results

bookmarkiconbase64 code
bookmarkiconbase64[+][X]

Click green "+" sign to add bookmarks to list for further steps. Then you can search other bookmarks and repeat adding to list until you're done.
Clicking red cross on the list below removes the bookmark from the list.

2. Choose icons

bookmarkiconbase64 code
bookmarkiconbase64[+][X]

You can either use the icon in the table or choose another, custom icon (remember, the size should be 16×16 px and the best format is PNG with transparent background!).
For icons from the table above just click the link [get code] right by the icon. (The data is provided by an external PHP script. If I wanted to include the function inside this extension it would require in consuming more of your computer resources and giving more permissions for this extesion to reach your data. I'd rather avoid this.)
In case of a bookmarklet or a custom icon or if the [get code] link doesn't work – click here.
For making custom icons I use MS Paint and then IrfanView or Gimp (for transparency). Then you may use one of the websites (still working in 2020) to convert the icon to base64:
  • pjhome.net – simply drag&drop downloaded icon/image onto their site (e.g. you can right-click one of the icons in the table above, save it (it'll show on the bottom bar of downloaded files – do not close it!), then go to that page and simply drag the icon file from the bottom bar and drop on the site)
  • base64-image.de – same as above
  • (there are plenty of websites that do the same work for you, for free!)
Then paste the result into the field in column base64 code of the row with the bookmark which you want to use with that icon. REMEMBER! The code should have the prefix data:image/png;base64, and finally look like data:image/png;base64,iVBORw0KGgoBlahBlahBlahLongLongLongWayLongerThanThis…
Repeat this step for each icon you want to change.



3. Save iconized bookmarks to file…

Right-click this link and choose "Save link as…" or "Save target as…". (Remember to save as HTML file, e.g. "bookmarks.html".)



4. …import that saved file…


NOTE: Unfortunately Google has removed the ability to import bookmarks from Chrome extensions. You have to do this manually by: 1) opening your Bookmark manager (Ctrl+Shift+O), 2) clicking menu (3 dots in upper right corner of the screen), 3) selecting "Import bookmarks" and choosing the HTML file you just saved.



5. …and remove duplicates

NOTE: Chrome imports new bookmarks to a folder at the far end of Bookmark bar. This folder's name is probably "Imported" or "Imported bookmarks" (maybe in your systems language). They are all duplicates of your bookmarks. All your original bookmarks are in their original places. But now they're with your new icons ;) You can simply remove the new folder (yes, with all imported bookmarks).
